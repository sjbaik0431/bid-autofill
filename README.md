# 입찰 정량평가 자동입력 시스템

대형 텐트/야외시설 렌탈 업체를 위한 **입찰 제안서 정량평가 양식 자동입력 시스템**입니다.

한컴오피스(HWP) 양식에 회사정보를 자동으로 입력하고 PDF로 변환합니다.

## 주요 기능

- **웹 대시보드** (Flask) - 브라우저에서 편리하게 조작
- **양식 파일 관리** - HWP 양식 업로드/삭제/선택
- **회사정보 자동추출** - 기존 HWP 파일에서 회사정보 자동 파싱
- **자동입력 엔진** - 한컴오피스 COM API로 찾아바꾸기 자동 실행
- **PDF 변환** - HWP → PDF 자동 변환
- **데모 모드** - 입력 과정을 화면에서 천천히 확인 가능

## 시스템 요구사항

- Windows 10/11
- Python 3.10+
- 한컴오피스 (한글 2018 이상)
- 필수 패키지: `flask`, `pywin32`, `olefile`

## 설치

```bash
pip install flask pywin32 olefile
```

## 실행 방법

### 방법 1: 대시보드 (권장)
```bash
python dashboard.py
```
또는 `대시보드 실행.bat` 더블클릭

브라우저에서 `http://localhost:5000` 접속

### 방법 2: CLI 실행
```bash
python autofill.py
```
또는 `autofill.bat` 더블클릭

## 사용법

### 1. 최초 설정
1. `company_master_sample.json`을 `company_master.json`으로 복사
2. 회사정보 수정 (업체명, 대표자, 주소 등)

### 2. 새 입찰 적용
1. **📁 양식 관리** 탭에서 새 양식 HWP 업로드
2. **🚀 자동입력 실행** 탭에서 입찰명/발주처/제출일 입력
3. 양식 선택 후 **▶ 자동입력 실행** 클릭

### 3. 다른 회사 정보 적용
1. **🏢 회사정보 관리** 탭에서 회사정보 HWP 업로드
2. 자동 추출된 정보 확인 후 **📥 반영** 클릭
3. **💾 회사정보 저장** 클릭

## 파일 구조

```
bid-autofill/
├── dashboard.py              # 웹 대시보드 서버 (메인)
├── autofill.py               # CLI 자동입력 스크립트
├── company_master.json       # 회사정보 DB
├── autofill_config.json      # 입찰별 설정
├── 대시보드 실행.bat           # 대시보드 런처
├── autofill.bat              # CLI 런처
├── 양식/                     # 업로드된 양식 파일 저장소
└── 회사정보/                  # 업로드된 회사정보 파일 저장소
```

| 항목 | 패턴 예시 |
|------|----------|

| 대표자 | `대 표 자 :`, `대표자 :`, `대표이사 :` |
| 주소 | `주소 :`, `소재지 :` |

| 전화/FAX | `전화번호 :`, `FAX :` |
| 자본금/매출액 | `자본금 :`, `매출액 :` |
| 날짜 | `2026년 월 일` → 제출일로 변환 |
