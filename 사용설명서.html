📋 입찰 정량평가 자동입력 대시보드

사용설명서 v1.0 · 2026년 4월

🔧 최초 설치 (PC당 1회만)

1Python 설치 확인
이미 설치되어 있다면 생략. 없다면 python.org에서 다운로드 → 설치 시 "Add Python to PATH" 반드시 체크
2한컴오피스 설치 확인
한/글 프로그램이 있어야 자동입력이 작동합니다. 미설치 시 설치 필요.
3바탕화면 바로가기 생성
프로그램 폴더에서 바탕화면 바로가기 만들기.bat 더블클릭 → 바탕화면에 📄 입찰 자동입력 아이콘 생성됨

▶️ 매일 사용법 (3단계)

1바탕화면 아이콘 더블클릭
📄 입찰 자동입력 아이콘 클릭 → 검은 창이 뜨고 브라우저가 자동으로 열림 (http://localhost:5000)
2정보 입력 & 실행
① 입찰명, 발주처, 제출일 입력
② 양식 파일 선택 (등록된 양식 또는 새 HWP 업로드)
③ [▶ 자동입력 실행] 버튼 클릭
④ 한글 프로그램이 자동으로 열리고 양식이 채워짐 (약 30초~2분 소요)
3결과 확인 및 종료
완성된 .hwp 및 .pdf 파일 생성 확인 → 검은 창에서 Ctrl+C로 종료

📑 대시보드 탭 기능

탭용도
🚀 자동입력 실행입찰명·발주처·제출일 입력 후 양식 자동채움 실행
🏢 회사정보 관리회사 기본정보(업체명·대표자·주소 등) 확인 및 HWP에서 자동추출
📁 양식 관리자주 쓰는 빈 양식 HWP 파일 업로드·삭제·관리
📋 실행 이력과거 자동입력 실행 기록 확인

⚠️ 주의사항

개인정보 보호
회사정보 HWP 파일에는 대표자 주민번호·전화번호 등이 포함될 수 있습니다. 외부 PC나 공유 폴더에 올리지 마세요.
한글 프로그램 중복 실행 금지
자동입력 중에는 한글 프로그램을 직접 조작하지 마세요. 이미 한글이 열려 있으면 미리 닫고 실행하는 것이 안전합니다.
양식이 새로 바뀌면?
📁 양식 관리 탭에서 새 HWP 업로드 → 자동입력 실행 시 선택. 라벨(예: "업체명 :")이 다르면 개발자에게 문의하여 패턴을 추가할 수 있습니다.

🔍 자주 묻는 문제

증상해결 방법
검은 창만 뜨고 브라우저가 안 열림브라우저에서 직접 http://localhost:5000 접속
"포트 5000 사용 중" 메시지대시보드가 이미 실행 중. 기존 창 확인
한컴오피스가 안 열림한글 2020 이상 필요. 관리자 권한으로 .bat 재실행
"Python이 없습니다" 오류python.org에서 Python 3.10+ 설치 (PATH 체크 필수)
패턴이 제대로 안 채워짐회사정보 탭에서 HWP 재추출 → 실행
